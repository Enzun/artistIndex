■ アーティスト追加の流れ

    【Admin フォームから1人追加】
    1. /admin を開く
    2. アーティスト名を入力して「追加」
    3. MusicBrainz + YouTube API + Wikipedia API が自動で走り、
       YouTube / Spotify / Wikipedia を芋づる式に取得してDBに登録される
    4. YouTubeが見つからない場合のみエラーになる（他は空欄のまま登録）

    【コマンドラインから一括追加】
    1. 名前リストのファイルを用意（1行1アーティスト名 or JSON配列）
    2. 実行:
           npx tsx --env-file=.env scripts/auto_add_artists.ts data/names.txt
           npx tsx --env-file=.env scripts/auto_add_artists.ts data/names.json
    3. YouTube未解決・エラーのアーティストはログに出るので手動確認

    【Spotify IDが未設定のアーティストを後から補完】
           npx tsx --env-file=.env scripts/fill_spotify_ids.ts
           npx tsx --env-file=.env scripts/fill_spotify_ids.ts --dry-run  # 確認のみ

■ アーティスト公開の流れ

    1. collecting 登録後、1〜2週間データを溜める
    2. Admin ページ（/admin）でアーティストを選んでグラフを確認
    3. 指数の動きが問題なければ「公開する」ボタンを押す → active に切り替わる
    4. /preview と / のアーティスト一覧に表示される

    ※ Admin パスワードは ADMIN_PASSWORD 環境変数で設定
